# mdl_icon
material design light icons from https://pictogrammers.com/library/mdil/ 


## Include 

```yaml
dependencies:
  flutter:
    sdk: flutter

  mdl_icons:
    git:
      url: https://github.com/m7mdra/mdl_icon
      ref: master

```

## Use


``` dart
    Icon(MaterialDesignLightIcons.alarm);
    //or
    Icon(MDL.alarm);
    //or
    Icon(MDLIcons.alarm);
```
